# Proposal: Restructure Phase 2 for AI-Driven Collaboration

## Goal
Simplify the coordinator workflow for Phase 2 (Student Assignment and Documentation) by unifying pages and introducing AI-powered bulk automation.

## Context
Currently, coordinators must jump between three different pages: Assignment List, Assignment Detail, and Nominal Register. Additionally, document management is manual and repetitive (3 docs per student).

## Proposed Changes
- **Unified Phase 2 Console**: Merge student selection and document management into a single, table-centric view.
- **AI Bulk Matcher**: Implement a "Magic Box" that accepts multiple PDFs, extracts IDALU/Name/Type, and auto-assigns them.
- **Admin "Turbo" Verification**: Upgrade the admin dashboard with AI confidence indicators and bulk approval actions.
- **Redundancy Removal**: Delete the old separate nominal register pages.

## Benefits
- Reduced friction for coordinators (less navigation).
- Drastic reduction in manual upload actions via AI matching.
- Faster verification cycle for administrators.
