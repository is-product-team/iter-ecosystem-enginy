# Proposal: Relax Teacher Requirements

## Context
Currently, the system forces centers to provide two "References" when requesting a workshop and two "Teachers" during the planning phase. 

## Problem
1. **Rigidity**: Some centers only have one coordinator for a specific workshop.
2. **Role Confusion**: The term "Referent" is used for both the pedagogical contact person and the actual workshop practitioner, leading to confusion.

## Proposed Solution
1. **Single Mandatory Referent**: Only one contact person will be required at the request stage.
2. **Role Separation**:
   - **Phase 1 (Request)**: Labeled as "Referent de Contacte" (Contact Reference).
   - **Phase 3 (Execution)**: Labeled as "Professor/a Responsable" or "Equip Docent" (Teaching Staff).
3. **Flexible Staffing**: Allow centers to assign only 1 practitioner in Phase 3 if desired.

## Success Criteria
- [ ] Centers can save requests with only one teacher.
- [ ] Labels in the UI clearly distinguish between "Contact" and "Staff".
- [ ] Phase 3 checklist "Designate Teachers" can be completed with only one teacher.
