# Proposal: Redesign Login and Registration Info

## Goal
To elevate the first impression of the Iter platform by modernizing the login page with a premium, Apple-inspired aesthetic and providing clear, localized registration instructions for different user roles.

## Context
The current login page is functional but feels slightly dated due to a rigid card layout and specific brand elements (like the blue divider) that break the minimalist flow. Additionally, users who don't have an account have no clear path to request access based on their role.

## Proposed Changes
1. **Modern Login**: A cleaner, softer layout removing high-contrast dividers and using refined typography.
2. **Registration Information**: A dedicated info view (accessible from the login page) that explains the onboarding process for Coordinators, Admins, and Teachers in a clear 3-column layout.
3. **Full Localization**: All new elements will be fully translated into Spanish, Catalan, English, and Arabic (with RTL support).
