# Specification: Mobile App

The mobile application is built with React Native and Expo, using NativeWind for styling.

## Architecture

- **App Router (`/app`)**: Expo Router for file-based navigation.
- **Components (`/components`)**: Mobile-optimized UI components.
- **Services (`/services`)**: Integration with the Backend API.

## Key Features

- **Attendance Tracking**: Marking students present/absent via mobile.
- **Telemetry**: Real-time GPS tracking for field activities.
- **Evaluation**: Mobile-first forms for teacher and student feedback.
- **Notifications**: Push notifications for workshop updates.
